Interface

  • Notwendige Features:
  • Tasks "Formular"
  • Input Elemente fuer ID, Type, data (input/output)
  • type: hash-md5 , hash-sha256
  • type: crack-md5
  • data: (String) input und (String) output

Beispiel

var json = {
	id: 0,
	type: 'hash-md5',
	data: {
		input:  'woot',
		output: null
	}
};
